## 第 5 页：列表内存布局对比（核心页）

**设计描述**：
```
顶部标题 "内存布局：列表 [1,2,3]"

左侧 Python：

- 文字："间接引用，每元素 36 字节"

右侧 Rust：
- 3 个紧密相邻的小方块，直接写 1、2、3
- 文字："直接存储，每元素 4 字节"

底部柱状图对比：
"100万元素：Python 56MB vs Rust 4MB = 14倍差距"
Python 长条，Rust 短条。
```

## 第 6 页：内存管理对比

**设计描述**：
```
顶部标题 "内存管理"

左侧 Python：
- 标题 "引用计数 + GC"
- 流程图：创建对象 → refcount+1 → 引用减少 → refcount-1 → 归零释放
- 额外分支：循环引用 → GC 扫描
- 底部标注："运行时开销"

右侧 Rust：
- 标题 "所有权系统"
- 流程图：创建 → 所有权转移（旧变量打叉失效）→ 离开作用域 → 自动释放
- 底部标注："零运行时开销，编译期保证"

两侧用对比色区分，中间竖线分隔。
```
